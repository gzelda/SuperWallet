>1:设置支付密码

1.请求地址: /login/setPayCode
2.请求方式: post
3.请求参数: UID user唯一id, payCode 支付密码
4.请求返回:
code\status | 0 | 1 |  2
---|---|---|---
0 | 请求失败
1 | 请求成功
2 | 系统异常
```
{
	"code":,		  0:fail 1:success 2:error
	"status":,		  
	"data":{

	}
}
```
