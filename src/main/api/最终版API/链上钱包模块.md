>1:链上钱包数据请求

1.请求地址: /dWallet/listDWalletInfo
2.请求方式: post

3.请求参数:
| 字段名 |  类型  | 说明  |
| :------- | :---- |:---- |
| UID |   String    |用户ID  |
4.请求返回:
code\status | 0 | 1 |  2
---|---|---|---
0 | 请求失败
1 | 请求成功
2 | 系统异常
```
{
	"code":,		  0:fail 1:success 2:error
	"status":,		  code的状态描述（表格）
	"data":{
        "tokenType": 0-int,  
        "tokenName":""-String, 
        "tokenAddress": ""-String,     
        "tokenPriceToRMB": 0-double
	}
}
```
