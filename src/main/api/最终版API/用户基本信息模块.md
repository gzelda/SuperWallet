code: 0.失败 1.成功 2.错误

>1:获取用户基本信息

1.请求地址: /user/token/{token}

2.请求方式: get

3.请求参数:
| 字段名 |  类型  | 说明  |
| :------- | :---- |:---- |
| token |   String    |cookie里的键token对应的值|

4.请求返回:

code\status | 0 | 1 |  2
---|---|---|---
0 | 登录已过期
1 | 用户基本信息返回成功
2 | 


```
{
    "code": 1,
    "status": 0,
    "msg":
    "data": {
        "uid":""-String,
        "nickname":""-String,
        "sex":0-byte,
        "isagency":0-byte,
        "phonenumber":""-String,
        "inviter":""-String,
        "status":0-byte,
        "invitedcode":""-String,
        "registertime":""-Date,
        "headphoto":  - byte[]
    }
}
```
